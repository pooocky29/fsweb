표시 방식

diplay:block;

  • 위에서 아래로 배치(세로 배치)
  • width, height, margin, padding, border가 모두 적용된다.
  • 기본값이 width:100%; height:콘텐츠크기;
  • html,body,div, h1~h6, p, ul, li(단,display가 block이 맞지만 list-item;아이템이다.), ...

display:inline;

  • 왼쪽에서 오른쪽으로 배치(가로 배치)
  • width와 height는 적용할 수 없다. (단, 이미지는 예외)
  • margin은 좌우만 적용된다.
  • padding, border는 모두 적용된다.
  • 기본값이 width:콘텐츠크기; height:콘텐츠크기;
  • span, img, a, em, b, i, label, ...
스팬 임시링크 강조 굵게 이텔릭

display:inline-block;

  • 왼쪽에서 오른쪽으로 배치(가로 배치)
  • width, height, margin, padding, border 모두 적용된다.
  • button은 box-sizing:border-box; 이다.(다른 블록요소/인라인요소들은 content-box임)
  • button, input, ...
